Working with drafts

New challenges start as drafts. Once you are happy with them, publish from the challenge page.

You can still run test cases while drafting, so feel free to iterate.

Tips for smooth releases

  • Include at least one sample case so players understand the task.
  • Balance points and time limits with the intended difficulty.
  • Write clear messages and notes so solvers never feel lost.
